# UPN Fundamentos de Algoritmos

Ejercicios del curso COMP1201A - 2026-1  
Universidad Privada del Norte

## Descripción
Este repositorio almacena ejercicios y ejemplos del curso, organizados por semanas y por lenguaje de programación.

## Estructura
Cada semana contiene:

- `csharp/`: ejercicios en C#
- `python/`: ejercicios en Python

Ejemplo:

- `semana04/csharp/`
- `semana04/python/`
